
# SIMPLE C.R.U.D

Exemplo de implementação de um CRUD (Create, Read, Update and Delete).

## Tecnologias e ferramentas utilizadas

**Backend**

- Node.js 
- MongoDB
> Express e Mongoose

**Frontend**

 - React.js
 - Styled Components
 - Eslint + Prettier para padronização de código
